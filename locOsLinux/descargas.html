32 bits (Beta)

La versión de 32 bits es ideal para resucitar algunos Pentium 4 y/o para cualquier PC con 1Gb de RAM.

Usa Kernel 4.9 LTS

Desacargar 32 bits

64 bits (Beta)

La versión de 64 bits es ideal para cualquier PC que tenga 2GB o más de RAM.

Además trae software adicional que su contraparte de 32 bits no trae, tal como Stremio, Steam y soporte a Flatpak.

Usa Kernel 5.10 LTS

Esta versión viene con blueman (programa para bluetooth) cargado al inicio, si no usas ningún dispositivo con bluetooth, puedes simplemente borrarlo desde la terminal con "sudo apt remove blueman". De esta manera economizarás aproximadamente 100mb de ram en el inicio (la versión de 32 bits no cuenta con blueman).
